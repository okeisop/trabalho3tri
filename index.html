Sobre o Projeto

O projeto “cultivando saberes” é uma iniciativa pedagógica que visa promover a educação ambiental e o desenvolvimento sustentável nas escolas. O projeto consiste na criação e manutenção de uma horta escolar, onde os alunos podem aprender sobre os ciclos da natureza, a produção de alimentos orgânicos, a conservação dos recursos naturais e a alimentação saudável. Além disso, o projeto busca estimular a integração entre as diferentes áreas do conhecimento, a participação coletiva e a responsabilidade social. O projeto “cultivando saberes” é uma forma de valorizar o meio ambiente e a cultura local, contribuindo para a formação de cidadãos conscientes e comprometidos com o futuro do planeta.
